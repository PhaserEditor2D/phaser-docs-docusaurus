---

In Phaser Editor you can encapsulate game object behavior into prefabs, [user components](../user-components), and [script nodes](../script-node). These mechanisms use the [user properties](../prefabs/prefab-user-properties) to customize the behaviors.

For a better implementation of behaviors related to [Spine](https://esotericsoftware.com) game objects, we provide two user property types:

* [Spine Skin Name property type](../prefabs/prefab-user-properties#spine-skin-name-property-type)
* [Spine Animation Name property type](../prefabs/prefab-user-properties#spine-animation-name-property-type)

In addition, the Event property type allows [Spine events](../prefabs/prefab-user-properties#dynamic-spine-events).
