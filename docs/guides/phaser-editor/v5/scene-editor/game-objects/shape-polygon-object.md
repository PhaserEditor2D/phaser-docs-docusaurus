You can create [Polygon](https://docs.phaser.io/api-documentation/class/gameobjects-polygon) object by dropping the **Polygon** block, from the Blocks view, on the scene.

![Create a polygon.](https://cdn.phaser.io/editor-docs/scene-editor-polygon-create-20220623.webp)

A polygon is made with points. You can edit the points in the **Polygon section** or with the [Polygon tool](../manipulation-tools#polygon-tool).

The Polygon section shows the Points field. The points are presented as a string of the X Y pairs of the points, separated by a space.

![Polygon section.](https://cdn.phaser.io/editor-docs/scene-editor-polygon-section-20220623.webp)

The Polygon tool (that you can activate with the `Y` key) shows the polygonâ€™s points in the scene. It allows you:

* Change the position of a point: drag the point with the mouse cursor.
* Delete a point: move the mouse cursor over the point and press the Delete key (or double click). When the cursor is over the point, it is painted in red color.

![Move polygon point.](https://cdn.phaser.io/editor-docs/scene-editor-polygon-edit-points-20220623.webp)
* Add a new point: move the mouse cursor over a line. It shows a white handler under the cursor, you can drag it for creating a new point.

![Add new point.](https://cdn.phaser.io/editor-docs/scene-editor-polygon-add-point-20220623.webp)![Add new point.](https://cdn.phaser.io/editor-docs/scene-editor-polygon-add-point-2-20220623.webp)

The [Polygon](https://docs.phaser.io/api-documentation/class/gameobjects-polygon) code is generated by the [scene compiler](../scene-compiler) using the [polygon object factory](https://docs.phaser.io/api-documentation/class/gameobjects-gameobjectfactory#polygon):

```javascript
const polygon = this.add.polygon(10, 10, "0 0 20 70 100 100 80 20");
```
