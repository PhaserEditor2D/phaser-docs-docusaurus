The [Phaser.GameObjects.Layer](https://docs.phaser.io/api-documentation/class/gameobjects-layer) object type allows grouping objects. Unlike the [Container](./container-object) objects, a layer cannot be transformed. It doesnâ€™t have position, angle or scale. Its only purpose is to sort the objects for rendering, quickly.

[Learn more about Layers in the Phaser docs](https://docs.phaser.io/api-documentation/class/gameobjects-layer).

[Learn more about working with parent objects in the Scene Editor](../working-with-parent-objects).

You can create a Layer by dropping a Layer built-in block on the scene,  or with the [Create Layer with Selection](#grouping-objects-in-a-layer) command.

![Creating an empty layer.](https://cdn.phaser.io/editor-docs/scene-editor-layer-drop-into-scene-12172020.webp)

The code generated by the [scene compiler](../scene-compiler), to create a Layer, is like this:

```javascript
// create the layer
const layer_1 = this.add.layer();

// create an object and add it to the layer
const someObject = ... ;
layer_1.add(someObject);
```


## Grouping objects in a Layer

You can create a layer with a couple of objects. You can do this by selecting the objects and executing the  [Create Layer with Selection](#grouping-objects-in-a-layer) command. The command is available in the [Parent context menu](../working-with-parent-objects#parent-context-menu).

When you execute the command, the selected objects are removed from the original parents and added to the new layer. All objects keep the same absolute position. If all objects are children of a common layer, then the new layer will be added to the original parent layer.
