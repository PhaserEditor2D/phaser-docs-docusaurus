The [Main menu](../workbench/main-menu) lists common general commands. You can open it by pressing in the menu button at the right of the [Main toolbar](../workbench/main-toolbar):

![Main menu.](../images/workbench-main-menu.webp)
* **Open Project**: launches the native directory chooser to open a new project.
* **Close Project**: closes the current project and returns to the [Start page](../first-steps/create-first-project).
* **Reload Project**: [reloads the project files](projects#reload-project).
* **Compile Project**: compiles all compilable files of the project. For example, the [scene](../scene-editor/scene-compiler) and [component](../scene-editor/user-components/user-components-compiler) files. This does not include TypeScript files or any other format with external compilers.
* **Open Visual Studio Code**: opens the project in a local [Visual Studio Code](https://code.visualstudio.com) instance. This is available only if the Phaser Editor server is running in [desktop mode](../misc/server-options). You can configure a different external editor. Check the **-external-editor-command** [server option](../misc/server-options).
* **Layout**: It shows a list of supported [window layouts](./layouts).
* **Select Color Theme**: opens the [Color Themes dialog](color-themes).
* **Command Palette**: opens the [Command Palette](command-palette).
* **Settings**: opens the [Settings dialog](./settings).
* **Check For Updates**: checks if there is a new version.
* **Phaser Explorer**: opens a new window with the [Phaser Explorer](https://explorer.phaser.io), an advanced tool to browse the Phaser API and examples.
* **Help**: opens this documentation in the browser.
* **About**: opens the About dialog.
