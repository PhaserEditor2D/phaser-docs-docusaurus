[Spine](https://esotericsoftware.com) is a popular and powerful editor for making 2D skeletal animations. It provides an [official runtime](https://esotericsoftware.com/spine-phaser) for Phaser.

The [SpineGameObject](spine-game-object) is a Phaser game object for display and control [Spine](https://esotericsoftware.com) animations.

The [Spine](https://esotericsoftware.com) Phaser runtime provides a rich set of features for loading, controlling and customizing the [Spine](https://esotericsoftware.com) animations in a Phaser game. All these features are available to you via the runtime API. In Phaser Editor, as a plus, we are including the basic and most important features into the visual tools. This is a gradual process, but the current set of [Spine](https://esotericsoftware.com) features supported by Phaser Editor will help you to integrate the [Spine](https://esotericsoftware.com) animations in your games and boost your productivity.
