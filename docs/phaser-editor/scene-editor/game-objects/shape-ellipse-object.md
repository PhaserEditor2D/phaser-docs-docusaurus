title: Ellipse
---

You can create an [Ellipse](https://docs.phaser.io/api-documentation/class/gameobjects-ellipse) by dropping the **Ellipse** block, from the Blocks view, on the scene.

![Create an ellipse.](https://cdn.phaser.io/editor-docs/scene-editor-ellipse-1-11092020.webp)

You can change the size of the Ellipse in the **Size section** or you can use the [Size tool](../manipulation-tools#resize-tool).

![Change the ellipse size.](https://cdn.phaser.io/editor-docs/scene-editor-ellipse-2-11092020.webp)

In addition, the Ellipse has the [Smoothness](https://docs.phaser.io/api-documentation/class/gameobjects-ellipse#smoothness) property (the number of points used when rendering it):

![Ellipse's smoothness property.](https://cdn.phaser.io/editor-docs/scene-editor-ellipse-3-11092020.webp)

The [Ellipse](https://docs.phaser.io/api-documentation/class/gameobjects-ellipse) code is generated by the [scene compiler](../scene-compiler) using the [ellipse object factory](https://docs.phaser.io/api-documentation/class/gameobjects-gameobjectfactory#ellipse):

```javascript
const ellipse = this.add.ellipse(10, 10, 100, 100);
```
