---
title: "Unlock the IDE"
---

By default, the IDE runs in a Free mode. This mode includes all the IDE features, but limits the number of files per project to 200. If you need to create bigger projects, you have to unlock the IDE.

To unlock the IDE:

1. Purchase a license key in the [Phaser Editor Gumroad page](https://gumroad.com/l/phasereditor/). That key is valid for all the IDE versions. If you have a valid license key from a previous version (v1 or v2) then you donâ€™t need to purchase a new one.
2. Click on the **Unlock Phaser Editor** option of the [Main menu](../workbench/main-menu). It opens a dialog to enter the license key:

![Unlock the editor.](https://cdn.phaser.io/editor-docs/first-steps-unlock-ide.webp)

There previous option is only available when the editor is running in [application mode](./install-server#application-mode), but you can open the terminal and write the command:

> ```javascript
> $ PhaserEditor.exe -lickey <license key>
> ```
